# 博客开张啦！

欢迎来到我的技术博客！

## 为什么搭建这个博客

这个博客主要用于记录：

1. **开发经验**: Unity 和 Cocos Creator 开发中的经验和技巧
2. **问题解决**: 遇到的技术问题和解决方案
3. **学习笔记**: 新技术的学习和总结
4. **个人思考**: 对技术和工作的一些思考

## 技术栈

本博客使用以下技术搭建：

- **Next.js 14**: React 框架，支持 SSR 和 SSG
- **TypeScript**: 类型安全的 JavaScript
- **MDX**: Markdown + React 组件
- **Tailwind CSS**: 实用优先的 CSS 框架
- **Vercel**: 免费部署平台
