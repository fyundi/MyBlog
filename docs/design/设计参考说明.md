# 博客设计参考说明

## 推荐的简约风格博客模板

### 1. **Medium 风格**
- **特点**：大量留白、清晰排版、专注阅读体验
- **广告位置**：文章中间、侧边栏
- **配色**：黑白灰为主，少量强调色

### 2. **Ghost 风格**
- **特点**：极简设计、优雅排版、响应式布局
- **广告位置**：文章顶部、底部、侧边栏
- **配色**：深色/浅色主题切换

### 3. **Dev.to 风格**
- **特点**：技术博客风格、卡片式布局、清晰分类
- **广告位置**：列表页、文章内嵌
- **配色**：浅色背景、彩色标签

### 4. **Hashnode 风格**
- **特点**：现代简约、阅读友好、社交功能
- **广告位置**：侧边栏、文章内嵌
- **配色**：白色背景、品牌色强调

## 简约风格设计原则

1. **大量留白**：内容周围有足够的空间
2. **清晰层次**：标题、正文、元信息层次分明
3. **简洁配色**：主色 + 强调色，不超过 3 种颜色
4. **易读字体**：选择易读的字体和合适的字号
5. **广告友好**：预留广告位，不影响阅读体验

## 广告布局最佳实践

1. **侧边栏广告**：右侧固定位置，不遮挡内容
2. **文章内嵌广告**：段落之间，自然融入
3. **列表页广告**：文章卡片之间，视觉统一
4. **响应式适配**：移动端使用横幅广告
