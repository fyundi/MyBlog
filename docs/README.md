# 项目文档

本目录包含项目的所有文档和说明文件。

## 📁 目录结构

### 📖 guides/ - 使用指南
包含项目的使用说明和部署指南：

- **`使用指南.md`** ⭐ - 项目完整使用说明（推荐先看）
- **`Vercel部署指南.md`** ⭐ - Vercel 部署完整指南（推荐）
- `广告集成示例.md` - 广告集成的示例代码

### 🎨 design/ - 设计参考
包含设计相关的文档：

- `设计参考说明.md` - 博客设计风格参考和最佳实践
- `UI优化建议.md` - UI 优化建议

### 📋 project/ - 项目文档
包含项目计划和总结文档：

- `项目计划书.md` - 项目初始计划和技术文档
- `实施总结.md` - 项目实施总结和已完成功能列表

### 📐 rules/ - 格式规范
包含文档格式和编写规范：

- **`markdown-format-guide.md`** ⭐ - Markdown 文档格式规范指南（编写文档前必读）

## 🚀 快速导航

### 新手入门
1. 查看 **`guides/使用指南.md`** - 了解如何使用项目
2. 查看 **`guides/Vercel部署指南.md`** - 学习如何部署到 Vercel

### 开发相关
- **编写文档**：查看 `rules/markdown-format-guide.md` ⭐ 必读
- **了解设计**：查看 `design/设计参考说明.md`
- **项目概览**：查看 `project/项目计划书.md` 和 `project/实施总结.md`

### 功能集成
- **广告集成**：查看 `guides/广告集成示例.md`

## 📝 文档维护

- 所有文档使用 Markdown 格式
- 编写新文档前请阅读 `rules/markdown-format-guide.md`
- 保持文档简洁、清晰、易于理解
